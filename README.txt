# IT Consulting Firm Website

## Project Overview
This project is a multipage, modern, and fully responsive website for an IT Consulting firm. It showcases the company’s services, team, portfolio of projects, and provides visitors with a contact form for inquiries. The website is built with semantic HTML5, styled using clean and modern CSS, and enhanced with basic JavaScript for interactivity such as responsive navigation and form validation.

The purpose is to create a professional online presence that effectively communicates the firm’s expertise and facilitates client engagement.

## Features
- Multipage structure: Home, About Us, Services, Portfolio, Contact
- Responsive design for seamless experience on mobile and desktop
- Clean, modern aesthetic with accent color highlights
- Interactive mobile navigation menu
- Basic contact form with client-side validation
- Consistent header and footer across pages for easy navigation
- Showcase of team members and client testimonials
- Portfolio gallery with project highlights

## Technologies Used
- HTML5 for semantic page structure
- CSS3 with Grid and Flexbox for modern responsive layouts
- JavaScript for menu toggle and form validation interactivity

## Installation and Usage
1. Clone or download the project to your local machine.
2. Open the root folder.
3. Open any HTML file (e.g., `index.html`) in a web browser to view the website.
4. Modify content, images, or styles in the respective folders as needed.
5. To deploy, upload the project files to any static web host or your preferred server.
